# StepWriter Shots Landing Page

Static GitHub Pages fallback landing page for StepWriter Shots.

## Files

- `index.html` — main landing page
- `styles.css` — page styling
- `assets/` — screenshots used by the page

## Publish with GitHub Pages

Create a public GitHub repository, add these files to the repository root, push to GitHub, then enable GitHub Pages from the repository settings using `main` branch and `/root`.
